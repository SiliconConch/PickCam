# 拾光相机 (PickCam) - 产品需求文档 v3.0

## 📋 项目概述

### 品牌定位

**中文名：** 拾光相机  
**英文名：** PickCam  
**核心理念：** 在时光流逝中,捕捉真实瞬间 - 不修饰,不美化,只记录  
**品牌联动：** 硅基拾贝公众号（拾荒者贝壳）

### 产品定位

**一句话定位：**  
学习SwanCam的克制美学与滤镜质量，加入智能水印和社交传播能力，成为"有思想的人"的影像记录工具。

**目标用户：**
- 硅基拾贝读者（天然种子用户）
- 追求真实记录的摄影爱好者
- 厌倦过度美颜的年轻人
- 认同"反算法美学"的创意人群

**完全自定义能力：**

### 2. 15个精品滤镜（SwanCam标准）

**设计理念：**
- 15个精品 > 100个平庸
- 摄影师手工调教
- 强调直出质量
- 滤镜强度可调（0-100%）

**滤镜列表：**
1. **原片** - 不做任何处理
2. **真实还原** - 矫正手机过度锐化
3. **胶片记忆** - 富士Superia 400风格
4. **黑白叙事** - 高对比度黑白
5. **清晨微光** - 柔和蓝调
6. **复古油画** - 油画质感，暖色
7. **柔光人像** - Kodak Portra 400
8. **城市街拍** - 高对比街头风
9. **自然鲜活** - 风景增强
10. **柔梦** - 梦幻柔焦
11. **电影感** - 电影宽容度
12. **暗调情绪** - 低调氛围
13. **粉调柔和** - 柔和马卡龙
14. **冷调清冽** - 高冷蓝调
15. **暖调怀旧** - 温暖回忆

**滤镜参数：**
- 亮度 (Brightness)
- 对比度 (Contrast)
- 饱和度 (Saturation)
- 色温 (Temperature)
- 色调 (Tint)
- 自然饱和度 (Vibrance)
- 颗粒 (Grain)
- 褪色 (Fade)
- 锐化 (Sharpen)

### 3. 盲盒文案系统 🎲

**100种内置文案：**
```
分类（各20个）：
- 时光系列：此时此刻、今日限定、平凡日子里的光...
- 生活系列：捕捉生活、真实日常、简单生活...
- 情绪系列：安静时刻、温柔的光、静谧瞬间...
- 观察系列：发现美好、细节之美、观察入微...
- 诗意系列：光影诗、时光诗篇、日子的诗意...
```

**核心特性：** ⭐
```
✅ 编辑功能：
- 可修改任何文案（包括默认文案）
- 修改后保存到个人配置

✅ 自定义功能：
- 添加无限制自定义文案
- 最长20字符
- 删除自定义文案

✅ 选择方式：
- 每日文案（同一时刻相同文案，仪式感）
- 随机文案
- 手动选择

✅ 数据存储：
- 本地存储（wx.storage）
- 不依赖云端
```

**设计理念：**
- 简短点缀，不画蛇添足
- 仪式感 > 随机性
- 可编辑 > 完全固定

### 4. 批量处理（防闪退优化）⚠️

**硬性限制：**
```
单次最多：
- 9张照片 OR
- 1个视频

超过限制：
- 前端直接拦截
- 提示用户分批处理
```

**防闪退机制：**
```javascript
关键措施：
1. 串行处理（逐张处理，不并发）
2. 间隔释放（每张间隔100ms）
3. 手动GC（wx.triggerGC）
4. 及时清理（临时文件立即删除）
5. Canvas复用（减少创建销毁）
6. 图片压缩（>2000px自动压缩）
7. 错误隔离（单张失败不影响其他）

处理流程：
for (let i = 0; i < photos.length; i++) {
  await processSinglePhoto(photos[i]);
  await sleep(100ms);  // 关键暂停
  wx.triggerGC();      // 手动GC
  cleanupTempFiles();  // 清理临时文件
}
```

**用户体验：**
- 实时进度显示（当前/总数）
- 处理完成后统计报告
- 成功数量 + 失败照片编号
