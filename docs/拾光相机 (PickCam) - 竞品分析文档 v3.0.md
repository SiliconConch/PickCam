# 拾光相机 (PickCam) - 竞品分析文档 v3.0

## 🏆 核心竞品：SwanCam（重点分析）⭐⭐⭐⭐⭐

### 产品背景

**开发者：** 摄影师个人开发  
**平台：** iOS  
**定位：** 专业摄影师的手机相机工具  
**口碑：** V2EX、即刻等社区高度推荐

### 核心优势

**1. 滤镜质量** ✅
```
特点：
- 15个左右精品滤镜（非100+平庸）
- 每个滤镜都经过手工调教
- 摄影师审美，非算法生成
- 直出质量极高，无需后期

质量标准：
- 真实还原（非过度修饰）
- 胶片质感（模拟真实胶卷）
- 色彩准确（符合摄影师审美）
```

**2. 产品哲学** ✅
```
克制：
- 功能少但精
- 没有花哨特效
- 没有美颜贴纸
- 专注摄影本质

极简：
- UI设计极简
- 操作流程简洁
- 无冗余功能
- 无广告干扰

专业：
- 支持RAW格式
- 精细参数控制
- 摄影师导向
```

**3. 用户口碑** ✅
```
V2EX等社区评价：
- "摄影师打造，质量有保证"
- "滤镜不多但每个都很好"
- "UI极简，用起来很舒服"
- "终于不用修图了"

目标用户：
- 摄影爱好者
- 追求品质的用户
- 厌倦花哨工具的人
```

### SwanCam的劣势

**功能单一：**
- ❌ 只有相机+滤镜
- ❌ 无水印功能
- ❌ 无批量处理
- ❌ 无文案系统

**缺少传播：**
- ❌ 无分享功能
- ❌ 无社区
- ❌ 无病毒传播机制

**数据价值：**
- ❌ 无数据统计

- ❌ 无长期价值积累
